Praktek Pemograman Web 1

Sesi 1

Praktek Pemograman Web 1 akan membahas tentang cara menampilkan halaman web dengan menggunakan HTML, CSS dan Javascript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pehamaman tersebut menuliskan hasil kreasinya sendiri.

  • HTML
  • CSS
  • Javascript

  1. Pengenalan HTML
  2. Tabel
  3. Form
